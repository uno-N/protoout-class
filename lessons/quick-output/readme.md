## 小さなアウトプットの意味合い

- 思考や感情等失われやすい情報も雑多に記録することで振り返りできる
- 継続することで自分だけの資産となる
- アウトプットの頻度を上げることで他者からのフィードバックの発生（アウトカム）頻度が上がる
